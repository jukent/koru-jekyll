# Contributing
We welcome contributions to improve the Koru Jekyll documentation and codebase. Here are some ways you can contribute:

## Reporting Issues
If you encounter any bugs or issues while using the Koru Jekyll theme, please report them to [ecs-web@ucar.edu](mailto:ecs-web@ucar.edu).

## Submitting Pull Requests
If you have improvements or new features to add, feel free to fork the repository and submit a pull request. Please ensure your code follows our coding standards and includes appropriate tests.
